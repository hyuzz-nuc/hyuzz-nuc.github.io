# 复古卡通风博客改造完整提示词
## 一、整体目标
将当前基于 Hugo + PaperMod 的个人博客，从简约风格改造为**复古卡通风（像素风+拟物UI）**，同时实现完整的功能生态，包括数据统计、文章管理、作者展示、互动体验等，兼顾实用性与风格统一性。

## 二、核心功能需求
### 1. 基础功能模块（必实现）
| 功能项 | 详细需求 |
|--------|----------|
| 网站访问人数 | 在页脚或侧边栏显示全站独立访客数（UV）和页面浏览量（PV），采用复古像素数字样式展示 |
| 阅读时长统计 | 1. 单篇文章显示预估阅读时间（保留现有功能）；2. 全站显示所有文章阅读时长总和；3. 统计数字适配复古风格 |
| 作者最近发表的文章 | 在首页/侧边栏突出显示最近3-5篇文章，包含标题、发布日期，模块样式为复古像素窗口 |
| 文章合集功能 | 1. 利用 Hugo 的 `series` 字段将多篇文章归类；2. 自动生成合集页面；3. 文章页显示所属合集链接，合集入口为像素风按钮 |
| 关于页（修复404） | 1. 创建 `content/about/index.md`，包含作者介绍、博客理念、联系方式；2. 关于页整体风格为复古卡通面板样式 |

### 2. 体验增强模块（高优先级）
| 功能项 | 详细需求 |
|--------|----------|
| 回到顶部按钮 | 1. 像素风箭头/红白机摇杆样式；2. 滚动时显示、顶部隐藏；3. 点击平滑回到顶部 |
| 文章目录折叠/展开 | 1. 目录模块改为复古窗口化设计；2. 带像素风“展开/收起”按钮；3. 默认折叠，点击展开 |
| 夜间/日间模式切换 | 1. 切换按钮改为复古像素开关（红白机按键样式）；2. 夜间模式适配像素星空背景；3. 保留自动跟随系统的逻辑 |
| 复古像素风访客计数器 | 替换普通数字统计，改为8位街机风格像素数字展示访问人数 |
| 文章阅读进度条 | 1. 页面顶部添加红白机加载条样式的进度条；2. 滚动时同步显示阅读进度；3. 适配复古色调 |

### 3. 互动趣味模块（中优先级）
| 功能项 | 详细需求 |
|--------|----------|
| 文章点赞/收藏功能 | 1. 像素风爱心/星星点赞按钮；2. 无需登录即可点赞；3. 基于 LeanCloud 存储点赞数；4. 按钮有点击动效（像素闪烁） |
| 随机文章按钮 | 1. 扭蛋机/街机摇杆样式按钮；2. 点击随机跳转一篇文章；3. 按钮悬停有复古震动效果 |
| 技术栈/兴趣标签墙 | 1. 在关于页添加像素风技术标签（Java、Spring Boot等）；2. 标签为拟物按钮样式；3. 支持点击筛选相关文章（可选） |

### 4. 实用优化模块（低优先级）
| 功能项 | 详细需求 |
|--------|----------|
| 全文搜索优化 | 1. 搜索框改为复古打字机/街机输入框样式；2. 搜索结果列表适配像素风排版；3. 保留原有搜索逻辑 |
| 移动端适配优化 | 1. 移动端自动切换简化版像素背景；2. 按钮放大适配手指操作；3. 目录改为侧边滑出式（复古游戏机菜单） |

### 5. 界面风格需求（复古卡通风）
- **整体视觉**：
  - 背景：像素风天空/草地/游戏场景，固定不随滚动；
  - 容器：页面主体/文章卡片/侧边栏添加木纹边框、圆角、阴影，模拟复古游戏窗口；
  - 色调：主色调以复古黄/红/蓝为主，贴合红白机/街机风格。
- **组件风格**：
  - 导航栏：像素风按钮，悬停有拟物按压效果；
  - 文章卡片：像素边框 + 复古像素字体标题；
  - 图标：所有社交/功能图标替换为像素风；
  - 按钮：交互元素（阅读更多、标签、合集入口）均为像素风设计。
- **字体**：
  - 标题：Press Start 2P/方正像素体等复古像素字体；
  - 正文：清晰易读的无衬线字体，微调为复古色调。

## 三、技术实现路径
### 1. 功能实现核心方案
| 功能类别 | 实现工具/技术 | 核心代码位置 |
|----------|---------------|--------------|
| 数据统计 | 不蒜子（Busuanzi）+ LeanCloud（点赞） | `layouts/partials/head.html`、`layouts/partials/footer.html` |
| 文章合集 | Hugo 原生 `series` 字段 | 文章 Front Matter、`layouts/series/list.html` |
| 关于页 | Hugo 页面创建 | `content/about/index.md`、`layouts/_default/single.html` |
| 最近文章 | Hugo 模板函数 | `layouts/index.html`、`layouts/partials/sidebar.html` |
| 复古样式 | 自定义 CSS + 静态资源 | `assets/css/custom.css`、`static/images/`（像素资源） |
| 交互功能（回到顶部/进度条等） | JS + CSS | `assets/js/custom.js`、`assets/css/custom.css` |
